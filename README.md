## Endpoints

### GET /livros

Retorna todos os livros.

### GET /livros/:id

Retorna um livro pelo ID.

### POST /livros

Cria um novo livro.

### PUT /livros/:id

Atualiza um livro existente.

### DELETE /livros/:id

Remove um livro.

## Validações

título não pode ser vazio
autor não pode ser vazio
ano precisa ser número maior que 0
